Para ejecutar el programa utilizar la siguiente sintaxis:

./s-init-env.sh <1-3> [1-2]

<1-3> Opción para elegir que es lo que se desea instalar o configurar en el
         habiente:
         1. Instalar todo el software necesario para nvim
         2. Instalar solo nvim y configurar su contenido de algunos plugins
         3. Configurar unicamente nvim

[1-2] Opción para seleccionar el administrador de paquetes
      1. apt
      2. dnf
